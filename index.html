Projects

FlappyBirdAndroid

FlappyBird clone implemented in C++ and ported to Android with the Android NDK. The graphics renderer is implemented with OpenGL ES 2.0 and the sound management is done through OpenSL ES. The level is procedurally generated.

You can see this project code here: https://github.com/danielperpar/FlappyBirdAndroid

You can download this project here: http://bit.ly/FBirdapk

MyFlappyBird

This is the PC version of the FlappyBird clone from the FlappyBirdAndroid project. This was the base C++ game used in the FlappyBirdAndroid project.

You can see a video here: https://youtu.be/rCdVIVTQ__8

You can see this project code here: https://github.com/danielperpar/MyFlappyBird

VRCameraWand

This is a Unity3D task for a job interview. It is a simulation of a VR wand. The user can pick up objects, rotate them and drop them either on a platform or in the scene. There is also a small puzzle to interact with.

You can see a video here: https://youtu.be/sMucSsK6jpc

You can see this project code here: https://github.com/danielperpar/VRCameraWand

You can download this project here: bit.ly/2KpX5NJ

Metalhalla

Final Project of the master's degree in Video Game Design and Development. Metalhalla is a 2.5D platformer game where you perform the role of a young viking fighting against a wicked sorcerer and his army of followers. The project was implemented using Unity3D. I was in charge of the game menus, boss implementation, player magical attacks and special effects.

Website of the project: https://metalhalla.carrd.co
